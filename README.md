## Описание приложения

**StoreProject** — мобильное Android-приложение интернет-магазина одежды. Позволяет просматривать каталог товаров с фильтрацией по категориям, выбирать размер и цвет, добавлять позиции в корзину и оформлять заказ.

## Реализованный функционал

- **Каталог товаров** — список из 10 позиций одежды с изображениями, названиями, описаниями и ценами.
- **Фильтрация по категориям** — Новинки, Джинсы, Футболки, Платья, Верхняя одежда, Худи.
- **Детальный просмотр товара** — модальный bottom sheet с фотографией, описанием, ценой и бейджем «NEW».
- **Выбор размера** — XXS, XS, S, M, L, XL.
- **Выбор цвета** — 6 вариантов (чёрный, белый, серый, коричневый, бежевый, тёмно-синий).
- **Корзина** — добавление товаров, управление количеством (увеличение / уменьшение), удаление отдельных позиций, полная очистка.
- **Подсчёт итоговой стоимости** — автоматически пересчитывается при изменении состава корзины.
- **Оформление заказа** — bottom sheet с подтверждением успеха или сообщением об ошибке при пустой корзине.
- **Нижняя навигационная панель** — переключение между экранами «Меню» и «Корзина», бейдж-счётчик товаров на иконке корзины.
